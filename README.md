# Простая планировка дома и смета

MVP Windows-программы на Python и PySide6 для рисования простой 2D-планировки дома, выбора материалов из JSON-каталога и автоматического расчёта примерной сметы.

## Установка

Нужен Python 3.11 или новее.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Если виртуальное окружение не используется, можно установить зависимость в текущий Python:

```powershell
python -m pip install -r requirements.txt
```

## Запуск

```powershell
python main.py
```

## Сборка EXE для Windows

Для сборки нужен Python и доступ к интернету для установки зависимостей.

Запустите:

```powershell
.\build.bat
```

Скрипт:

- установит зависимости из `requirements.txt`;
- соберёт программу через PyInstaller;
- создаст папку `dist\Планировка дома и смета`;
- скопирует туда `materials.json`, `prices.json`, `README.md` и `app.ico`, если иконка есть.

Готовый файл программы будет здесь:

```text
dist\Планировка дома и смета\Планировка дома и смета.exe
```

## Сборка установщика

Для установщика нужен Inno Setup.

1. Сначала соберите EXE:

```powershell
.\build.bat
```

2. Откройте `installer.iss` в Inno Setup.
3. Нажмите `Compile`.

Готовый установщик появится в папке:

```text
installer\
```

Установщик:

- ставит программу в `Program Files`;
- создаёт ярлык в меню `Пуск`;
- по выбору создаёт ярлык на рабочем столе;
- после установки предлагает сразу запустить программу.

## Что отправлять пользователю

Самый простой вариант для человека без Python - отправить или скачать готовый архив:

```text
house-planner-windows.zip
```

Пользователь должен распаковать архив и запустить файл:

```text
Планировка дома и смета\Планировка дома и смета.exe
```

Если архив собирается вручную, отправляйте всю папку:

```text
dist\Планировка дома и смета
```

Если нужен обычный установщик, отправляйте файл из папки:

```text
installer\
```

Не отправляйте один `.exe` отдельно от папки `dist`, потому что PyInstaller используется в режиме `onedir` и рядом с exe лежат необходимые файлы приложения.

## Как пользоваться

- Дом автоматически вписывается в рабочее поле при открытии проекта.
- **Вписать проект в экран**: нажмите кнопку сверху или слева, чтобы план занял большую часть холста.
- **Масштаб**: вращайте колесо мыши над планом.
- **Перемещение холста**: зажмите среднюю кнопку мыши и двигайте план.
- **Быстро вписать**: дважды нажмите средней кнопкой мыши по холсту.
- **Добавить стену**: нажмите `Добавить стену`, затем протяните мышью линию на сетке.
- Во время рисования рядом с временной линией сразу показывается текущая длина стены.
- **Выбрать / двигать**: выберите инструмент `Выбрать / двигать`, кликните по стене и перетащите её.
- **Изменить стену**: выберите стену. Справа появятся длина, высота, толщина, материал, цена за м² и чекбокс `Несущая стена`.
- При изменении толщины стены линия на плане становится тоньше или толще визуально.
- **Добавить дверь**: нажмите `Добавить дверь`, выберите шаблон или `Свой размер`, затем кликните по существующей стене.
- **Изменить дверь**: выберите дверь. Справа можно менять шаблон, ширину, высоту, цену, направление открывания и сторону петель.
- **Добавить окно**: нажмите `Добавить окно`, выберите шаблон или `Свой размер`, затем кликните по существующей стене.
- **Изменить окно**: выберите окно. Справа можно менять шаблон, ширину, высоту, высоту от пола, стеклопакет, цену и количество в смете.
- **Режим крыши**: нажмите `Крыша`, чтобы включить показ крыши, подсветить её элементы на плане и открыть вкладку `Крыша` справа.
- **Два этажа**: во вкладке `Дом` выберите `2 этажа`. Сверху появится вкладка `План 2 этажа`.
- **Создать 2 этаж**: нажмите слева `Создать 2 этаж на основе 1 этажа`. Программа скопирует наружные несущие стены первого этажа, а внутренние перегородки, окна и двери второго этажа можно рисовать отдельно.
- **Помещение**: нажмите `Помещение`, кликните внутри контура и выберите назначение комнаты. На плане появится подпись с названием и площадью.
- **Лестница**: во вкладке `Помещения` задайте тип, ширину, длину, высоту подъёма и количество ступеней, затем нажмите инструмент `Лестница` и кликните на плане.
- **Фасад**: откройте вкладку `Фасад` над рабочим полем и выберите сторону: север, юг, запад или восток.
- **Разрез**: откройте вкладку `Разрез`, чтобы увидеть фундамент, цоколь, этажи, перекрытие, стены и крышу сбоку.
- **Экспликация**: откройте вкладку `Экспликация`, чтобы увидеть таблицу помещений по этажам и общую площадь.
- **Удалить объект**: выберите стену, дверь или окно и нажмите `Удалить выбранное` либо клавишу `Delete`.
- Сверху есть современная панель: `Проект`, `Конструкции`, `Планировки`, `Смета`, `Экспорт`.
- Слева есть быстрые кнопки `Новый проект`, `Открыть`, `Сохранить`, `Экспорт сметы` и `Вписать проект в экран`.

## Что видно на плане

- Стены рисуются тёмными толстыми линиями.
- Выбранная стена подсвечивается оранжевым.
- Дверь отображается как проём в стене, полотно двери и дуга открывания на 90 градусов.
- Окно отображается как проём в стене с двумя синими параллельными линиями.
- Выбранная дверь подсвечивается оранжевой рамкой.
- Выбранное окно подсвечивается голубой рамкой.
- Помещения отображаются зелёной подписью с названием и площадью.
- Лестница отображается отдельным строительным символом со ступенями и стрелкой направления подъёма. Выбранная лестница подсвечивается зелёным.
- В режиме крыши показываются внешний контур свеса, внутренний контур стен, конёк, стрелки скатов, фронтоны и подписи размеров.

## Правая панель

Правая панель сделана карточками с крупными русскими подписями. Она разделена на вкладки:

- `Дом` — этажность, высота этажей, цоколь, перекрытие, материал новых стен, фундамент, утепление и фасад;
- `Стены` — параметры выбранной стены;
- `Крыша` — тип крыши, угол, конёк, свес, фронтон, кровля, 3D-просмотр и расчёт крыши;
- `Окна` — параметры выбранного окна;
- `Двери` — параметры выбранной двери;
- `Помещения` — параметры выбранной комнаты и настройки лестницы;
- `Смета` — общая смета, площади этажей, лестница, перекрытие и отдельные строки по крыше.

При выборе стены, окна, двери или крыши программа автоматически открывает нужную вкладку.

## Этажи, помещения и лестница

Этажность теперь хранится как реальные планы этажей, а не только как коэффициент в смете.

- При режиме `1 этаж` отображается только вкладка `План 1 этажа`.
- При режиме `2 этажа` отображаются `План 1 этажа` и `План 2 этажа`.
- У каждого этажа свои стены, окна, двери, помещения и лестницы.
- Кнопка `Создать 2 этаж на основе 1 этажа` копирует несущие стены первого этажа на второй.
- Инструмент `Помещение` создаёт подпись комнаты на плане и добавляет её в `Экспликацию`.
- Инструмент `Лестница` ставит лестницу на план, добавляет её стоимость в смету и показывает в разрезе.

Расчёт площади помещения в этой версии практичный: для прямоугольных помещений программа ищет ближайшие стены вокруг клика. Если контур сложный, используется приблизительная площадь этажа.

## Крыша

Крыша вынесена в отдельную вкладку `Крыша` и работает как управляемый объект проекта.

Можно настроить:

- тип крыши: плоская, односкатная, двускатная, вальмовая, полувальмовая, мансардная или шатровая;
- направление конька: по X или по Y;
- материал кровли: металлочерепица, профлист, мягкая кровля, фальцевая кровля, керамическая черепица, цементно-песчаная черепица или ондулин;
- угол наклона крыши;
- высоту конька;
- свес крыши;
- высоту фронтона;
- коэффициент сложности;
- автоматический расчёт высоты конька для двускатной крыши;
- режим `Показать крышу на плане`;
- видимость отдельных слоёв: конёк, скаты, свесы, размеры, помещения, окна и двери.

Кнопка `Автоматически построить крышу` подбирает типовую схему по внешнему контуру дома. Для прямоугольных домов включается простая двускатная крыша, для более сложного контура - упрощённая вальмовая. Это быстрый старт, который потом можно поправить вручную.

Расчёт крыши использует площадь контура стен, свес, коэффициент типа крыши, коэффициент угла, коэффициент отходов кровли и коэффициент сложности. Для двускатной крыши при включённом авторасчёте высота конька считается по формуле `height = (house_width / 2) * tan(angle)`.

Во вкладке `Крыша` отдельно показываются:

- тип крыши;
- угол;
- высота и длина конька;
- свес;
- площадь кровли;
- площадь каждого ската;
- материал кровли;
- примерный вес кровли;
- срок службы материала;
- стоимость кровли;
- площадь фронтонов;
- стоимость фронтонов;
- общая стоимость крыши.

В режиме показа крыши поверх плана появляется полупрозрачный внешний контур со свесом. Крыша строится автоматически по контуру дома и не рисуется как обычные стены:

- для плоской крыши показывается только внешний контур;
- для двускатной крыши показывается толстая линия конька, фронтоны и стрелки направления скатов;
- для односкатной крыши показывается стрелка уклона;
- для вальмовой и шатровой крыши показываются диагональные линии от углов к коньку.

Внизу блока крыши есть легенда:

- сплошная линия = конёк;
- пунктир = свес;
- стрелка = направление ската;
- заливка = скат крыши.

В блоке крыши также есть простой интерактивный `3D-просмотр`: плоская крыша отображается плитой, односкатная - наклонной плоскостью, двускатная - двумя скатами, вальмовая - упрощённой моделью с четырьмя скатами. Кнопка `Открыть просмотр крыши` открывает отдельное большое окно, где модель можно вращать мышью и масштабировать колесом.

## Фасад

Во вкладке `Фасад` показывается простой 2D-вид выбранной стороны дома:

- север;
- юг;
- запад;
- восток.

На фасаде отображаются стена выбранной стороны, окна, двери, высота этажей, уровень пола, цоколь, перекрытие, крыша, конёк, свесы и размеры окон/дверей. Если выбран режим `2 этажа`, фасад показывает окна и двери обоих этажей. Это не 3D и не инженерный чертёж, а понятный строительный вид сбоку для быстрой проверки.

## Разрез

Во вкладке `Разрез` отображается простой строительный разрез:

- фундамент;
- цоколь;
- стены;
- этажи;
- перекрытия;
- лестница;
- крыша;
- высота конька.

Разрез показывает размерные линии высоты 1 этажа, перекрытия, 2 этажа, конька и общей высоты дома. Он нужен для быстрой проверки общей высоты и пропорций дома.

## Сохранение и экспорт

В верхней панели `Проект` доступны:

- `Новый проект`;
- `Открыть`;
- `Сохранить`;
- `Сохранить как`.

В верхней панели `Экспорт` доступны:

- `PDF`;
- `Изображение`;
- `Коммерческое предложение`.

В меню `Файл` также оставлены команды старого формата: открыть/сохранить JSON и экспортировать смету в TXT или PDF.

Старые JSON-проекты с полем `openings` продолжают открываться: двери и окна автоматически переводятся в новый формат.

## Каталог материалов

Основной каталог материалов находится в файле `materials.json`. Его можно менять без переписывания программы.

В каталоге есть разделы:

- `wall_materials` — газоблоки разной толщины и каркас;
- `brick_types` — виды кирпича;
- `block_types` — пеноблок, керамзитоблок, арболит, теплоблок и шлакоблок;
- `roof_types` — типы крыш и их коэффициенты;
- `roofing_materials` — материалы кровли, вес, отходы, срок службы и сложность монтажа;
- `foundation_types` — типы фундамента;
- `flooring` — цена перекрытия за м²;
- `stairs` — базовые расценки лестницы;
- `insulation_types` — утепление;
- `facade_finish` — фасадная отделка;
- `window_templates` — шаблоны окон и вариант `Свой размер`;
- `door_templates` — шаблоны дверей и вариант `Свой размер`.

`prices.json` оставлен только для старой совместимости. Новые списки материалов, размеры, характеристики и цены берутся из `materials.json`.
