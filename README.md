# Простая планировка дома и смета

MVP Windows-программы на Python и PySide6 для рисования простой 2D-планировки дома и автоматического расчёта примерной сметы.

## Установка

Нужен Python 3.11 или новее.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Если виртуальное окружение не используется, можно установить зависимость в текущий Python:

```powershell
python -m pip install -r requirements.txt
```

## Запуск

```powershell
python main.py
```

## Сборка EXE для Windows

Для сборки нужен Python и доступ к интернету для установки зависимостей.

Запустите:

```powershell
.\build.bat
```

Скрипт:

- установит зависимости из `requirements.txt`;
- соберёт программу через PyInstaller;
- создаст папку `dist\Планировка дома и смета`;
- скопирует туда `prices.json`, `README.md` и `app.ico`, если иконка есть.

Готовый файл программы будет здесь:

```text
dist\Планировка дома и смета\Планировка дома и смета.exe
```

## Сборка установщика

Для установщика нужен Inno Setup.

1. Сначала соберите EXE:

```powershell
.\build.bat
```

2. Откройте `installer.iss` в Inno Setup.
3. Нажмите `Compile`.

Готовый установщик появится в папке:

```text
installer\
```

Установщик:

- ставит программу в `Program Files`;
- создаёт ярлык в меню `Пуск`;
- по выбору создаёт ярлык на рабочем столе;
- после установки предлагает сразу запустить программу.

## Что отправлять пользователю

Если установщик не нужен, отправляйте всю папку:

```text
dist\Планировка дома и смета
```

Если нужен обычный установщик, отправляйте файл из папки:

```text
installer\
```

Не отправляйте один `.exe` отдельно от папки `dist`, потому что PyInstaller используется в режиме `onedir` и рядом с exe лежат необходимые файлы приложения.

## Как пользоваться

- **Добавить стену**: нажмите `Добавить стену`, затем протяните мышью линию на сетке.
- Во время рисования рядом с временной линией сразу показывается текущая длина стены.
- **Выбрать / двигать**: выберите инструмент `Выбрать / двигать`, кликните по стене и перетащите её.
- **Изменить стену**: выберите стену. Справа появятся длина, высота, толщина, материал, цена за м² и чекбокс `Несущая стена`.
- При изменении толщины стены линия на плане становится тоньше или толще визуально.
- **Добавить дверь**: выберите `Добавить дверь` и наведите мышь на стену. На стене появится предпросмотр места установки. Клик ставит дверь.
- **Изменить дверь**: выберите дверь. Справа можно менять ширину, направление открывания и сторону петель.
- **Добавить окно**: выберите `Добавить окно` и кликните по существующей стене.
- **Изменить окно**: выберите окно. Справа можно менять ширину, высоту и количество окон в смете.
- **Режим крыши**: нажмите `Крыша`, чтобы включить показ крыши на плане и перейти к настройкам крыши справа.
- **Удалить объект**: выберите стену, дверь или окно и нажмите `Удалить выбранное` либо клавишу `Delete`.
- Слева есть быстрые кнопки `Новый проект`, `Открыть`, `Сохранить` и `Экспорт сметы`.

## Что видно на плане

- Стены рисуются тёмными толстыми линиями.
- Выбранная стена подсвечивается зелёным.
- Дверь отображается как проём в стене, полотно двери и дуга открывания на 90 градусов.
- Окно отображается как проём в стене с двумя синими параллельными линиями.
- Выбранная дверь или окно подсвечиваются пунктирной рамкой.

## Правая панель

Правая панель меняется в зависимости от выбора:

- если ничего не выбрано, показываются общие параметры проекта;
- если выбрана стена, показываются параметры стены;
- если выбрана дверь, показываются параметры двери;
- если выбрано окно, показываются параметры окна.

Смета находится ниже параметров и обновляется после каждого действия.

## Крыша

В параметрах проекта есть отдельный блок `Крыша`.

Можно настроить:

- тип крыши: плоская, односкатная, двускатная или вальмовая;
- направление конька: по X или по Y;
- материал кровли: металлочерепица, профлист или мягкая кровля;
- угол наклона крыши;
- высоту конька;
- свес крыши;
- коэффициент сложности;
- режим `Показать крышу на плане`.

Расчёт крыши использует площадь контура стен, свес, коэффициент типа крыши, коэффициент угла и коэффициент сложности. В смете показываются тип крыши, материал кровли, примерная площадь крыши и стоимость крыши.

В режиме показа крыши поверх плана появляется полупрозрачный внешний контур со свесом. Крыша строится автоматически по контуру дома и не рисуется как обычные стены:

- для плоской крыши показывается только внешний контур;
- для двускатной крыши показывается линия конька;
- для односкатной крыши показывается стрелка уклона;
- для вальмовой крыши показываются диагональные линии от углов к коньку.

В блоке крыши также есть простой `3D-просмотр`: плоская крыша отображается плитой, односкатная - наклонной плоскостью, двускатная - двумя скатами, вальмовая - упрощённой моделью с четырьмя скатами.

## Сохранение и экспорт

В меню `Файл` доступны:

- `Новый проект`;
- `Открыть JSON...`;
- `Сохранить JSON...`;
- `Экспорт сметы TXT...`;
- `Экспорт сметы PDF...`.

Старые JSON-проекты с полем `openings` продолжают открываться: двери и окна автоматически переводятся в новый формат.

## Как менять цены материалов

Все базовые цены лежат в файле `prices.json`.

Если `prices.json` удалить, программа создаст его заново с базовыми ценами при следующем запуске.

В файле хранятся:

- `wall_materials` — цена стеновых материалов за м² стены;
- `foundation` — цена фундамента за м² площади дома;
- `roofing` — цена кровли за м² условной площади крыши;
- `roof_type_multiplier` — коэффициент типа крыши;
- `doors` — цена двери;
- `windows` — цена окна;
- `finishing` — цена отделки за м² площади дома.

## Что уже есть

- рабочее поле с сеткой;
- рисование, выделение, перемещение и удаление стен;
- визуальная толщина стены зависит от параметра `Толщина стены`;
- архитектурные символы дверей и окон;
- отображение длины стены во время рисования;
- предпросмотр двери или окна при наведении на стену;
- параметры стен, дверей и окон в правой панели;
- отдельный режим, параметры, 2D-контур и простой 3D-просмотр крыши;
- выбор материалов стен, фундамента, крыши, кровли и отделки;
- автоматический расчёт площади, длины стен и стоимости;
- сохранение и открытие проекта в JSON;
- экспорт сметы в TXT и PDF.

## Что можно добавить потом

- точный расчёт площади по замкнутым комнатам, а не по габаритному прямоугольнику;
- перетаскивание уже установленной двери или окна вдоль стены;
- отдельные помещения с названиями;
- разные типы дверей и окон с разными ценами;
- несколько этажей на отдельных вкладках;
- печать плана вместе со сметой;
- упаковку в `.exe` через PyInstaller.
